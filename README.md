- automated testing
- linting and formatting
- Git hooks
- GitHub Actions CI
- AI governance
- documentation structure
- reproducible workflows

# Setup

## Create virtual environment and install tooling

    make setup

## Run Tests

    make test

## Run Linting

    make lint

## Run Formatting

    make format

## Install Git Hooks

    pre-commit install
